**Some Links, Tips & Thoughts for the day**
**Electron**
  I have been trying out the Electron JavaScript framework, very interesting the technology as it takes JavaScript out of
  the browser and on to the desktop working on Windows, MacOS and Linux. I am currently putting together some notes in
  the course notes repo here on my Github account.
 
 
      https://github.com/marb61a/Course-Notes/tree/master/Javascript/Electron
 
 
  The technology itself is very very interesting as Electron packages an entire version of Chromium with the app which
  gives the bonus that there is no need to worry about vendor prefixes in development. Electron gives you everything 
  both a client side and a server side application gets which makes it very important in the age of cross platform
  application development
  
  
**Laravel**
  I have been continuing using Laravel, the very popular web development framework using PHP, my experience so far has been
  immensely positive. Anybody who is familiar with Django, Rails etc will find this to be very easy to use. I am also
  happy that it uses Vue.js for its view library, this is a less complex React like library and it is experiencing huge
  growth on Github. This was developed by taking the best of other frameworks and putting it together, this is a tutorial
  for anyone wondering how to get started
 
 
      https://scotch.io/tutorials/build-a-to-do-app-with-vue-js-2
  
  
**Which CSS**
  There are a plethora of CSS frameworks, libraries, tools, techniques and layout models available for anybody developing 
  a site or sites, this is without custom
    
  - https://foundation.zurb.com/sites.html                                  -- Foundation 6
  - https://v4-alpha.getbootstrap.com                                       -- Bootstrap
  - https://css-tricks.com/snippets/css/a-guide-to-flexbox/                 -- Flexbox
  - http://sass-lang.com                                                    -- Sass
  - http://postcss.org                                                      -- PostCSS
  
  These are only some of the options available without thinking of things such as SkeletonCSS and mobile frameworks such
  as Framework 7. This comes against a backdrop of increasing complexity on the Frontend, I personally like Bootstrap 4 as it
  nearly always works without issues, has a new grid system and has retired its Glyphicon library and instead embraced Font
  Awesome's excellent collection of icons.
  
  PS
  This is without talking about CSS in JS which is a new way of styling components

  NPM
  When running npm script there are a range of flags that can be passed to change the level of log output
  
  
      -- loglevel silent or  --silent or -s will reduce log output considerable 
      -q for quiet
      -d for info
      -dd for verbose
      -ddd for hugely verbose
